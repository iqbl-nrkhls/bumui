# BUMUI

BUMA UI Library

## Instalation

To install dependencies:

```bash
# using NPM
npm install bumui-vue

# using Yarn
yarn add bumui-vue

# Using Bun
bun install bumui-vue
```

# Usage

```js
import "bumui-vue/style.css";
import { BButton } from "bumui-vue";

<BButton text="Button" primary />;
```
